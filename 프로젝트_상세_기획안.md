# 📊 AI-Inv: 글로벌 매크로 분석 파이프라인 명세서 (v5.0 Snapshot)

이 문서는 AI 기반 투자 비서(AI-Inv) 시스템의 **글로벌 5대 매체 확장 및 1시간 단위 추적 아키텍처**를 완전히 반영한 최종 기획 명세서입니다.

---

## 1. 프로젝트 개요
**AI-Inv**는 Google Sheets를 중앙 '데이터 웨어하우스'로 활용하고, GitHub Actions 스케줄러를 통해 하루 2번 글로벌 증시와 경제 속보를 자동으로 긁어 모아(Bulk Crawling), **최신의 스냅샷(Snapshot) 상태를 유지**하는 데이터 파이프라인입니다. 최종적으로 사용자의 구글 드라이브(Gemini Gems) 프롬프트 구조와 연동하여 시장 요약 및 투자의견을 브리핑받는 1인 컨트롤 타워 역할을 수행합니다.

## 2. 주요 기능 및 파이프라인 단계

### Phase 1: 스케줄러 기동 (GitHub Actions)
- `cron: '0 23,7 * * 1-5'` 규칙에 따라 평일 최적의 시간(개장 전후 등)에 서버리스 파이썬 스크립트(`main.py`)가 구동됩니다.
- 무거운 글로벌 데이터를 안전하게 다루기 위해 **트랜잭션 구조**가 적용되어, 수집에 성공했을 때만 구글 시트에 업데이트 요청(Bulk Insert)을 보냅니다.

### Phase 2: Crawler Engine 가동
총 3개의 거대한 크롤링 엔진이 동작하며 총 120개 전후의 RAW 텍스트 행을 긁어모읍니다.

#### (A) Expert Analytic 엔진 (블로그 전문 수집)
- **대상**: 네이버 파워블로거(ranto28)
- **방식**: 기존의 단순 웹스크래핑을 넘어 **공식 RSS 피드(`rss.blog.naver.com`)**를 추적.
- **결과**: 최신 글 20건의 고유 URL을 획득 후, 각 페이지에 다이렉트 접속해 본문 **전문(Full Text)**을 추출.

#### (B) Market Indicators 엔진 (타임머신 추적)
- **대상**: 글로벌 증시(M7, S&P500), 금리/채권, 환율, 원자재, 암호화폐, 국내 주요 우량주 등 **총 50개 매크로 핵심 지표** (+ CNN Fear & Greed Index).
- **방식**: `yfinance` 라이브러리의 시계열 기능(`interval="1h", period="3mo"`) 활용. CNN 지수는 API 폴링.
- **결과**: 현재 종가(Value)와 함께 **"1h, 2h, 6h, 12h, 24h, 7d, 30d, 60d 전" 대비 변동폭(%) 문자열 배열**을 생성하여 단/중기 추세와 변동성 동시 추적.

#### (C) News Calendar 엔진 (글로벌 100 뉴스 봇)
글로벌 매크로 이슈를 선점하기 위해 **5대 뉴스 채널에서 각 20건씩(최대 100건) 본문 텍스트 통째로 수집**.
- **국내 소스**: 네이버 금융(경제/해외 메인), 매일경제(MK) 증권망.
- **해외 소스**: CNN Business(RSS 기반), Reuters(구글 뉴스 RSS 우회망), Financial Juice(터미널식 최신 단문 헤드라인 20건 추출).
- **결과**: 뉴스 링크 클릭 없이 즉시 AI가 파악할 수 있도록 본문을 모두 긁어(`BeautifulSoup <p>` 태그 기반) RAW 데이터를 구성.

### Phase 3: Uploader (Snapshot & Bulk Insert)
- **Snapshot 초기화**: 시트 용량 초과 및 구동 과부하 방지를 위해, 과거 분량을 무한정 밑으로 누적하지 않습니다. `clear_sheet_data()` 함수가 **기존 데이터 영역을 완전히 삭제(백지화)합니다.**
- **Bulk Insert**: 구글 API의 Rate Limit 한계(가장 악명높은 차단 요인)를 회피하기 위해 한 줄씩 보내지 않고, **확보한 100개의 데이터를 2차원 배열로 뭉쳐 한 번의 트랜잭션으로 삽입**(`insert_multiple_rows_top`)합니다.

### Phase 4: AI 분석 및 브리핑 (Gemini)
- Dashboard가 업데이트 되면서 빈 화면에: `"Gemini AI: 이 셀에 Market_Indicators와 News_Calendar 탭을 읽고 글로벌 시장 요약을 입력해줘."` 라는 **AI 전용 트리거 프롬프트**를 남겨둡니다.
- 유저가 구글 드라이브(Gemini)에 접속하면 해당 프롬프트를 인식해, 백지화 기반으로 갱신된 최신 120개 데이터들을 읽고 종합 보고서를 산출합니다.

## 3. 구조 요약 (탭 구성)
1. **Dashboard**: 업데이트 시간 (최신), 현재 수집 상태(Success/Fail), Gemini AI 분석용 전용 프롬프트 지시어 삽입 공간.
2. **Expert Analytic**: 날짜 / 출처 / 제목 / **본문** / 링크 (20건 스냅샷)
3. **Market Indicators**: 날짜 / 지수명 / **현재가** / 시그널 / **시간별/일자별 변동폭(%) 내역** (50건 스냅샷)
4. **News Calendar**: 날짜 / 매체(카테고리) / 제목 / 중요도 / **본문 전문** (최대 100건 스냅샷)

---

## 4. 보안 및 유지보수
- **Secrets Management**: GitHub 스케줄러 환경 변수에 `GOOGLE_SERVICE_ACCOUNT_JSON`, `SPREADSHEET_ID` 암호화 저장.
- **Fault Tolerance**: 로이터 등 글로벌 뉴스 서버에서 크롤러 봇 접속을 401 코드로 차단할 경우, 이를 회피하기 위해 구글 뉴스 RSS의 Description(요약본)으로 우회 백업 데이터를 수집하여 멈추지 않고 흘러가게 설계했습니다.
- **Extensibility**: 추후 새로운 지수(예: 비트코인) 또는 채널이 추가될 시 `finance_scraper.py`에 모듈(Function) 단위로 가볍게 끼워 넣을 수 있습니다.
